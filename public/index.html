Introduction

VelyDocs Anime REST API

API scraping untuk sumber anime Indonesia. Mendukung 3 sumber: Samehadaku (anime sub Indo), Anichin.me (anime), dan Anichin.id (donghua/anime China).

Base URL: loading...

Semua endpoint membutuhkan API key. Tambahkan ?apikey=YOUR_KEY atau header X-API-Key: YOUR_KEY.

Response format: JSON. Semua response memiliki field status, creator, dan data.

Features

• Anti-block system dengan rotating User-Agent, random delay, dan smart caching
• Full pagination support di semua endpoint yang mendukung
• Episode streaming servers dan download links
• Search, genre filter, A-Z list, schedule, popular ranking
• Response caching (5 menit) untuk performa optimal

Authentication

API Key Required — Semua endpoint dilindungi API key. Akses tanpa key atau key salah akan ditolak. Mau akses? Beli key-nya.

Cara Penggunaan

Query Parameter:
GET /api/samehadaku/home?apikey=YOUR_KEY

Header:
X-API-Key: YOUR_KEY

Response tanpa key:

{
  "status": false,
  "code": 401,
  "message": "Lu siapa? Mau akses tanpa key? Mimpi kali. Beli dulu key-nya.",
  "creator": "Gxyenn"
}

GET Samehadaku - Home

Ambil data homepage: anime terbaru dan ranking popular.

/api/samehadaku/home

GET Samehadaku - Recent

Ambil daftar anime terbaru dengan pagination.

/api/samehadaku/recent?page={page}

GET Samehadaku - Search

Cari anime berdasarkan keyword. Mendukung pagination.

/api/samehadaku/search?q={query}&page={page}
ParameterTypeRequiredDescription
qstringYesSearch keyword
pagenumberNoPage number (default: 1)

GET Samehadaku - Anime Detail

Ambil detail anime lengkap: info, genre, sinopsis, daftar episode.

/api/samehadaku/anime/{slug}
ParameterTypeRequiredDescription
slugstringYesAnime slug (contoh: one-piece)

GET Samehadaku - Episode

Ambil detail episode: streaming servers, download links, navigasi.

/api/samehadaku/episode/{slug}
ParameterTypeRequiredDescription
slugstringYesEpisode slug (contoh: one-piece-episode-1159-subtitle-indonesia)

GET Samehadaku - Genres

Ambil daftar semua genre yang tersedia.

/api/samehadaku/genres

GET Samehadaku - Genre List

Ambil daftar anime berdasarkan genre. Mendukung pagination.

/api/samehadaku/genre/{slug}?page={page}
ParameterTypeRequiredDescription
slugstringYesGenre slug (contoh: action)
pagenumberNoPage number (default: 1)

GET Samehadaku - A-Z List

Browse anime berdasarkan huruf alfabet.

/api/samehadaku/az-list?letter={letter}&page={page}
ParameterTypeRequiredDescription
letterstringNoHuruf filter (A-Z, kosong = semua)
pagenumberNoPage number

GET Samehadaku - Schedule

Ambil jadwal rilis anime per hari.

/api/samehadaku/schedule

GET Samehadaku - Popular

Ambil daftar anime populer dari sidebar ranking.

/api/samehadaku/popular

GET Samehadaku - Anime List

Browse semua anime dengan filter status, type, dan sorting.

/api/samehadaku/anime-list?page={page}&status={status}&type={type}&order={order}
ParameterTypeRequiredDescription
pagenumberNoPage number (default: 1)
statusstringNoongoing, completed, upcoming
typestringNotv, movie, ova, ona, special
orderstringNoupdate, title, titlereverse, popular, rating (default: update)

GET Anichin - Home

Ambil data homepage Anichin.me: trending dan latest episodes.

/api/anichin/home

GET Anichin - Explore

Browse katalog anime dengan pagination, sorting, dan filter alfabet.

/api/anichin/explore?page={page}&sort={sort}&letter={letter}
ParameterTypeRequiredDescription
pagenumberNoPage number (default: 1, max ~74)
sortstringNolatest_update, popular, rating, title
letterstringNoFilter huruf (A-Z)
